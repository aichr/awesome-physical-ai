# Contributing to Awesome Physical AI

Thank you for your interest in contributing!

## How to Contribute

### Adding a Resource

1. **Search first** - Make sure the resource isn't already listed
2. **Choose the right section** - Papers, Frameworks, Datasets, Simulators, Projects, etc.
3. **Include required information:**
   - Title/Name
   - Link (must be working)
   - Brief description (1-2 sentences)
   - (For papers) Publication venue/year if applicable

### Categories to Add To

- **Papers**: Academic papers (arXiv, conference proceedings)
- **Frameworks**: Libraries, tools, agent frameworks
- **Datasets**: Robot datasets, embodied AI datasets
- **Simulators**: Physics simulators, research environments
- **Projects**: Notable robot projects, demos
- **Courses**: University courses, tutorials
- **Community**: Forums, newsletters, events

### Quality Guidelines

- ✅ Working links
- ✅ Accurate descriptions
- ✅ Relevant to Physical AI
- ✅ No spam or self-promotion

- ❌ Broken links
- ❌ Duplicate entries
- ❌ Unverified/unsafe content

## Pull Request Process

1. Update the relevant section
2. Keep alphabetical order where applicable
3. Use consistent formatting
4. Describe your changes clearly

## Questions?

Open an issue for discussion before submitting large changes.
