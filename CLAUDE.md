## What this project contains

- **Papers**: Latest research in Physical AI, robot learning, embodied AI
- **Frameworks**: Libraries and tools for building Physical AI systems
- **Datasets**: Robot manipulation, navigation, and embodied AI datasets
- **Simulators**: Physics simulation environments for robotics
- **Projects**: Notable robot projects and demos
- **Courses**: University courses and tutorials
- **Community**: Forums, newsletters, conferences

## How to maintain this list

1. **Keep it current** - Physical AI moves fast; update monthly
2. **Verify links** - Check that URLs still work
3. **Balance** - Include both academic and practical resources
4. **Categorize** - Put new items in the right section
5. **Describe** - Brief description for each item (1-2 sentences)

## Key resources to monitor

- arXiv:cs.LG, cs.RO, cs.CV (robotics, ML papers)
- GitHub trending: physical-ai, embodied-ai, robot-learning topics
- Conferences: CoRL, ICRA, IROS, RSS, NeurIPS
- Hugging Face: LeRobot, robot learning models
